🚀 Overview
AgroTech addresses the "yield gap" faced by Malaysian smallholders by synthesizing real-time environmental data with Gemini 3.1 Flash. 
The system provides hyper-localized "Actionable Instructions" to help farmers shift from reactive labor to proactive data-driven management.

🛠️ Tech Stack
UI Framework: Streamlit 
Intelligence Engine: Google Gemini 3.1 Flash 
Environmental Data: Gemini API
Backend: Python
